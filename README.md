# Mandelbrot_Set
これはマンデルブロ集合をレンダリングするために作られたプロジェクトです。

## 実行方法
cargoをインストール済みならお馴染みの以下のコードで実行する事ができますよ。
```
cargo run
```
あるいは
```
cargo run --bin mandelbrot_set
```
でもOKです。

コンパイルしようとすると7Gbyteぐらいのファイルが生成されるので時間がかかるかもしれません。

## 動作環境
以下の環境での動作が確認されました。
Windows x86-64 Nvidia GEFORCE RTX

GPUを使うため環境によっては動作に大きく違いが現れるかもしれません。（最悪の場合、動きません）

## 実行後は…
ウィンドウが作成されるはずです。

矢印キーで画面を移動したり、
Zキーでズームイン、Xキーでズームアウトすることができます。
F11キーでフルスクリーンにすることもできます。（同じキーで解除することもできます）

## 設定
プロジェクトを始めて実行するとconfig/config.txtが生成されます。
中身は以下のようになっているはずです。
```
execution = 20;
create_report = true;
```

パラメータ"execution"はマンデルブロ集合をレンダリングする際の反復回数を意味しており、
数値が高くなればなるほどより繊細になります。（その分処理が重くなります）
有効な値は 10 | 20 | 50 | 100 | 200 | 500 | 1000 | 2000 | 5000 のうちのいずれかとなります。

パラメータ"create_report"はシステム終了後、レポートを生成するか否かを示しており、
trueは生成、falseは生成しません。

実行時、設定が何らかの理由により読み込めない（誤字や有効な値ではない）場合、
デフォルトの設定が読み込まれます。
